# 🛰️ **СПУТНИКОВЫЙ МОНИТОРИНГ 90-Х**  
### *Terminal Interface v2.1 — «ORBIS-97»*  
> _«Связь установлена. Орбита стабильна. Система — АКТИВНА»_

---

## 📡 Основная концепция

**«Спутниковый мониторинг 90-х»** — интерактивный веб-интерфейс, имитирующий промышленный терминал эпохи позднего 90-х:  
- 🔧 *Техно-эстетика*: вдохновлено АСУ ТП, NASA-дисплеями, **Warhammer 40k** (Tech-Priest HUD-стиль) и первыми web-сайтами  
- 🌌 *Функционал*: мониторинг орбитальных аппаратов с симуляцией реального времени  
- 🧠 *Цель*: ностальгия + обучение + эстетическое погружение

> ⚠️ **Важно**: Интерфейс *не подключен* к реальным системам NORAD/Space-Track. Данные — симулированы для демонстрации.

---

## 🎯 Цель проекта

| Аспект           | Реализация                                                                 |
|------------------|----------------------------------------------------------------------------|
| **Историческая реконструкция** | Табличная верстка, `<marquee>`, «в разработке», счётчики Geocities        |
| **Образование**  | Визуализация Кеплеровых орбит, связи, энергоснабжения                     |
| **Интерактивность** | Анимированные спутники, TTS-оповещения (опц.), кликабельные объекты      |
| **Ностальгия**   | Пиксельная графика, dial-up boot sequence, «System Initializing...»       |
